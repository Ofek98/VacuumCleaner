\doxysubsection{Detailed Description}
Implementation file for the \mbox{\hyperlink{classHouse}{House}} class. 

This file contains the implementation of the \mbox{\hyperlink{classHouse}{House}} class and its member functions. The \mbox{\hyperlink{classHouse}{House}} class represents a house with a matrix of tiles, where each tile can have a status indicating the amount of dirt or other properties such as walls or docking stations. The \mbox{\hyperlink{classHouse}{House}} class provides functions to access and manipulate the tiles in the house.
